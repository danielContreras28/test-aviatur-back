## Run Locally

Clone the project

```bash
  git clone https://github.com/danielContreras28/test-aviatur-back
```

Go to the project directory

```bash
  cd test-aviatur-back
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
